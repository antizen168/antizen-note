### 学习内容

```tip Code
- python解析器
    - python版本（2.x 3.x）
    - python书写格式
    - python执行方式（脚本文件执行、交互方式执行）
    - 常用的Python工具（eclipse/PyCharm/visio studio Code）
- 数据类型
    - 数值（整数/浮点数/布尔值）
    - 字符串（如：'123'）
    - 列表（如：[1,2,3]）
    - 元组（如：(1,2,3)）
    - 字典（如: {'a':1,'b':2}）
    - 集合（如：{'a','b','c'}）
    - 空 （None 相当于java中的null）
- 变量与运算符
    - 变量的命名规则
    - 变量赋值
    - 运算符
- 流程控制
    - if/elif/else
    - for/for 变量 in 列表
    - continue（跳出本次循环）
    - break（跳出循环）
    - pass（什么都不做）
- 输入输出
    - 输出 print("字符串") / print("字符串",Object) / print(Object)
    - 输入 input()
- 函数和模块
    - 函数定义（def 函数名(参数):）
    - 函数调用（函数名(参数)）
    - 函数返回（return）
    - 函数参数（关键字参数、可变长参数、变量作用域）
    - 迭代器（next()、 iter()、list()）
    - 生成器（yield）
    - 高阶函数（filter()、map()、zip()、reduce()）
    - 闭包（闭包的定义、闭包的使用、装饰器）
    - 模块的导入（import 模块 / from 模块 import 子模块、函数）
- 面向对象
    - 类的定义
    - 类对象
    - 继承
    - 常用的内置方法
        - __init__(self): 初始化方法，类似于构造器
        - __str__(self): 对象格式化方法
        - __add__(self,other): 运算符重载方法
    - 内置属性
        - __dict__ : 类的属性（包含一个字典，由类的数据属性组成）
        - __doc__ :类的文档字符串
        - __name__: 类名
        - __module__: 类定义所在的模块
        - __bases__ : 类的所有父类构成元素（包含了一个由所有父类组成的元组）

```
