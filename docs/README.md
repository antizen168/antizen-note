## Getting Started

```bash
# 安装vuepress组件，如已安装，就不需要安装此组件
npm install -g vuepress

# 导出Antizen项目
git clone https://github.com/forwardfirst/antizen-note.git

# 本地开发环境查看
npm run dev

# 本地编译
npm run build

# 部署至github
npm run deploy:win /  npm run deploy:linux

```
